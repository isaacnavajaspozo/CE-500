##.üßë‚Äçüéì Explicaci√≥n de componentes OpenVass:
	gvm-libs:		            	Biblioteca C utilizadas en openvas-scanner , gvmd , gsad y pg-gvm.
	gvmd:			              	Servicio principal, utiliza PostgreSQL y realiza autenticaci√≥n, gesti√≥n de escaneos, la informaci√≥n sobre vulnerabilidades y mucho m√°s
	pg-gvm:			            	Extensi√≥n de PostgreSQL que agrega varias funciones utilizadas por gvmd, (iCalendar y evaluaci√≥n de rango de host)
	GSA:			              	La aplicaci√≥n web en JavaScript, React y NodeJS
	gsad:			              	En C. Se comunica con gvmd mediante GMP. Servidor web con contenido est√°tico y proporciona una API para la web
	openvas-smb:		        	M√≥dulo auxiliar para openvas-scanner, para interactuar con sistemas Microsoft Windows a trav√©s de la API
	openvas-scanner:	      		Motor de escaneo con todas las funciones que ejecuta un flujo de pruebas de vulnerabilidad (VT) continuamente actualizado y ampliado
	ospd-openvas:		        	Motor de escaneo con todas las funciones que ejecuta un flujo de pruebas de vulnerabilidad (VT) continuamente actualizado y ampliado
	openvasd:		            	Utiliza para detectar productos vulnerables, actualmente solo notus est√° integrado en gvmd
	greenbone-feed-sync:	  		script basado en Python para descargar todos los datos de la fuente de la comunidad Greenbone a su m√°quina local
	gvm-tools:		          	Colecci√≥n de herramientas que ayudan a controlar instalaciones 
	servidor Redis:		      		Utilizado por el esc√°ner (openvas-scanner y ospd-openvas) para manejar la informaci√≥n de VT 
	Est√°ndares relacionados:
		[NVT]:		            	Es el conjunto de pruebas de vulnerabilidades de la comunidad Greenbone
		[CERT]:	 	            	Incluye avisos de seguridad de organizaciones como CERT-Bund o DFN-CERT
		[GVMD_DATA]: 	        	Contiene configuraciones necesarias para que OpenVAS funcione correctamente, como Policies, Port Lists, Report Formats, Scan Configs.
		[SCAP]:	Est√°n dividido en:
				+ [[CVEs]]:     (Common Vulnerabilities and Exposures): Base de datos de vulnerabilidades conocidas.
				+ [[CPEs]]:     (Common Platform Enumeration): Identificadores √∫nicos para sistemas operativos y software.
